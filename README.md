# EEE 213: Introduction to Logic

A 14-week digital logic course built around one method (design everything as a truth table) and
one goal (assemble a working 4-bit microcontroller), using the browser-based LogicLab simulator.

Live site: https://senolgulgonul.github.io/logic/

Built with Jekyll (the `jekyll-theme-cayman` theme via `_config.yml`); GitHub Pages renders the
Markdown to HTML automatically.
